7. By XPath

A very flexible, but sometimes brittle way to find elements. Can navigate through the DOM.

Paragraph 1

Paragraph 2

Some text
'XPath'Find'//div[@id="xpathContainer"]/p[1]' (first paragraph)
'XPath'Find'//span[contains(text(), "text")]' (partial text)
'XPath'Find'//*[@id="xpathContainer"]//span' (any descendant)
'XPath'Find'//button[@type="submit"]' (finds the submit button)

8. Handling Hidden/Non-Interactable Elements

This element is hidden initially.

Find'hiddenElement' (will find, but might not be visible/interactable initially)
Click'toggleHiddenButton' (to make it visible)
wait.Until(ExpectedConditions.ElementIsVisible(By.Id("hiddenElement) (for robust tests)
